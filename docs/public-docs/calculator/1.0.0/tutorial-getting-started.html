Tutorial: Getting Started

Getting Started

Import the module like so:

var Calc = require('/path/to/calc');

Add

var calculator = new Calc();
calculator.add(1, 3); // 4

Subtract

var calculator = new Calc();
calculator.subtract(2, 1); // 1
